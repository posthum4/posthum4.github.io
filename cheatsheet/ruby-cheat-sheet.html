Cheat Sheet: Ruby

Ruby Basics

Class:

A class is a collection of properties and traits shared by a group of objects. All classes are also objects in Ruby.

Object:

Everything in Ruby is an Object. Objects represent a single instance of something in Ruby. It can be an integer, string, array, or an entire class.

String:

Strings are text in Ruby. A string can be an empty string, represented by "", or a full page of text.

Integer:

Ingeters are whole numbers. They do not include values after a decimal place

Array:

Arrays are an ordered collection of items. This items can anything including other object, arrays, strings, etc. The position is indexed starting at 0. Zero is the first item in the array.

Hash:

Hashes are very similar to arrays. They contain a collection of items. Unlike arrays they are indexed by Keys. Keys are unique values in the hash.
Basic Ruby Method

to_s, to_i

123.to_s #=> "123"

"123".to_i #=> 123


On the first line, 123 is converted into a string.

On the second line, "123" is coverted into an integer.

nil?

nil.nil? #=> true

"nil".nil? #=> false


First line, nil? is to check if nil is nil.

Second line, "nil" is a string with actual value, .nil? return false.

gets.chomp and gets

puts "Input a string"

input = gets.chomp #=> "string"

"Input a string"

input = gets #=> "string\n"


First line, chomp will make Ruby to ignore the last character input, which is the return key

Without the chomp method, the return key is captured as \n

include?

a = [ "a", "b", "c" ]

a.include?("b") #=> true

a.include?("z") #=> false


include? can detect if a particular value is included in an array.

split

"1234".split("") #=>[1, 2, 3, 4]

"1, 2, 3, 4".split(",") #=>[1, 2, 3, 4]


split method took an argument, can split a string by the argument and turns it into an array

length

a = [1, 2, 3, 4]

a.length #=>4


length method can return the number of element in an array
Looping in Ruby

While Loop:

While loops are used when you do not know how many times you will need to repeat a specific block of code. As long as the conditon holds it will continue to run what is in the code block
while condition do
  puts ("Keep printing this while true")
end
Variations include: Until, or Begin end while.

For Loops:

For loops are used when you know how many times you need to perform a specific action.
for i in 0..5
  puts ("Print this for the #{i}")
end
